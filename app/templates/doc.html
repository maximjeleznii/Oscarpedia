Oscarpedia Documentation

The Oscarpedia is a database of Oscar winning movies from the years 2000-2007. It allows users multiple ways to look through the movies, find them on IMDB and give their own personal rating. Each movie entry features the Title, Year, Poster, Description, IMDB Rating, Oscar Awards, and a rating given by Oscarpedia Users.

GET /Oscarpedia

  • provides a GUI endpoint for the the first page of all the results of the database, initially paginated to the first 5 results.

GET /Oscarpedia/json

  • provides a JSON endpoint for the first page of all the results of the database.
>
Queries for both /Oscarpedia and /Oscarpedia/json
?page=<Page Number> Any integers 0-24 are valid, returns that page of results. A page of 0 or no page for /Oscarpedia/json will return all results. Pages are paginated to 5 results each.
?search=<Search> Any characters are valid, returns result based matching characters from any field.
?year=<Year> Any characters are valid, characters other than integers likely won’t return results. Returns result based matching characters from the year field.
?name=<Search> Any characters are valid, returns result based matching characters from the title field.
?description=<Search> Any characters are valid, returns result based matching characters from the description field.
?oscars=<Search> Any characters are valid, returns result based matching characters from the oscars field.
?rating=<Search> Any characters are valid, characters other than integers likely won’t return results. Returns result based matching characters from the rating field.
?user_rating=<Search> Any characters are valid, characters other than integers likely won’t return results. Returns result based matching characters from the user rating field.

GET /Oscarpedia/json/<id>

  • Provides a JSON endpoint to any movie in the database based on the movie’s id in the database. Id will return valid results for any integer between 1 and 120.
